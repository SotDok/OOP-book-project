# 📚 My Library

A simple JavaScript library management app that allows users to add, display, toggle read status, and delete books. Built with **Vanilla JS** and styled most of it with **Tailwind CSS**.

---

## Features

- Add new books with **title, author, pages, and read status**.
- Display all books in a **clean card layout**.
- Toggle the **read status** of each book.
- Delete books from the library.
- Books are stored in a JavaScript array (`myLibrary`) for simplicity.
